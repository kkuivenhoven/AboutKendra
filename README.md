# About Kendra
This website provides links to deployed projects that I created for my personal portfolio.

## Getting Started
1. Clone the repo
2. $ rake db:migrate db:seed
3. $ rails s

Or you may visit the web app [here](https://aboutkendra.herokuapp.com).
